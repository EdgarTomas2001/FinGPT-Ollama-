# FinGPT-Ollama-
FinGPT ist ein KI-gestütztes Trading-System für MetaTrader 5. Es kombiniert RSI, MACD, Support/Resistance und Multi-Timeframe-Analyse mit Ollama-Modellen. Features: Auto-Trading, Risk-Management, Logging und flexibles Menü – entwickelt für praxisnahes, sicheres Algorithmic Trading.
