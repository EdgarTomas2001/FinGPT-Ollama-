# Dokumentation - FinGPT

Hrein findest du die vollständige Dokumentation zum FinGPT-System.

### Funktionen
- RSI & MACD Analyse
- Dynamische Support & Resistance Levels
- Ollama LLM Integration (lokal)
- Automatischers Handel via MT5

### Installation
1. Ollama installieren und Modelle laden: `fingpt`, `llama3`, `mistral`
2. Python VENV erstellen und Pakete installieren:
- `pip install metatrader5`
- `pip install pyyaml`
- `pip install onnxruntime`(falls nötig)

### Konfiguration
Editiere das `Config.yaml`, um die Verbindung zu MT55 herzustellen.
