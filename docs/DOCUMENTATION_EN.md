# Documentation - FinGPT

Here you will find the full documentation for the FinGPT system.

### Features
- RSI & MACD Analysis
- Dynamic Support & Resistance Levels
- Ollama LLM Integration (local)
- Autoated trading via MT5

### Installation
1. Instalm Ollama and pull `pingpt`, `llama3`, `mistral`
2. Create Python VENV and install packages:
- `pip install metatrader5`
- `pip install pyyaml`
- `pip install onnxruntime`(if needed)

### Configuration
Edit the `config.yaml` to establish connection to MT5.

### Disclaimer
This is an experimental project. Trade at your own risk.
